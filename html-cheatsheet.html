HTML Cheat Sheet

This page is my personal HTML reference. It includes common tags and how I use them.

# Element / Feature Description Example
1 <h1>â€“<h6> Headings, used for titles. <h1>Main Title</h1>
2 <p> Paragraph of text. <p>Hello world</p>
3 <a> Link to another page or website. <a href="https://example.com">Click</a>
4 <img> Displays an image. <img src="image.jpg" alt="Description">
5 <ul> & <li> Unordered list. <ul><li>Item</li></ul>
6 <ol> & <li> Ordered (numbered) list. <ol><li>First</li></ol>
7 <div> Block-level container for content. <div class="box">Content</div>
8 <span> Inline container, often used for styling part of text. <span class="highlight">text</span>
9 <header> & <footer> Semantic elements for top and bottom of a page. <header>Site Title</header>
10 <nav> Section of page with navigation links. <nav><a href="#">Home</a></nav>
11 <section> Logical section of content. <section>About me</section>
12 <article> Independent content like a blog post. <article>Post</article>
13 <table>, <tr>, <td> Tabular data. <table><tr><td>Cell</td></tr></table>
14 <form>, <input>, <button> Collect user input. <form><input type="text"><button>Send</button></form>
15 <link> & <script> Include CSS and JavaScript files. <link rel="stylesheet" href="styles.css">
